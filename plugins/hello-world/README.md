# hello-world

A friendly hello-world plugin for Claude Code — part of the [oh-my-powers](https://github.com/zhengmingming/oh-my-powers) marketplace.

## Skills

- **greet** — Greets the user and introduces oh-my-powers. Triggered when you say hello or ask what this plugin does.

## Usage

After installing from the marketplace:

```
/plugin install hello-world@oh-my-powers
```

Then just say "hello" or "what can this plugin do?" and the greet skill will respond.
